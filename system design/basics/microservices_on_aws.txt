# Microservices on AWS

EC2

[ Amazon API Gateway ]
        |
        |
        |                       [ store/get     ]       -> [ Amazon EC2(m5.large)   ]       -| 
[ ElasticLoadBalancer ]      -> [ store/post    ]       -> [ Amazon EC2(t3.micro)   ]       -|  Common databases / insolated databases
                                [ store/delete  ]       -> [ Amazon EC2(m4.2xlarge) ]       -|
